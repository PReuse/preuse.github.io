  • HDFS Tiered Storage *

    The HDFS Tiered Storage-project makes it possible for HDFS to fetch blocks (parts of files) from external FileSystems or Cloud Storage and deeply integrate it into the core of Hadoop. This will, amongst having other features, make it possible for Hadoop to run on data stored in an external Cloud Storage System and store its results directly on the same Cloud, without needing two manual file copies by system administrators.
    For this project, I worked closely together with colleagues at Amplidata (an HGST- and Western Digital subsidiary) and Software Engineers of Microsoft.

    05/2016 - 01/2017

  • S3A, an S3-FileSystem for Hadoop *

    S3A is the recommended S3-client for Hadoop 2.7 and later.
    A successor to the s3n:// filesystem, the S3A:// system uses Amazon's libraries to interact with S3. This allows S3A to support larger files, higher performance operations and more. S3A has been usable in production since Hadoop 2.7, and is undergoing active maintenance for enhanced security, scalability and performance.

    09/2015 - 01/2017
